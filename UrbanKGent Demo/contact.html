UrbanKGC Service

UrbanKGent is a online agent providing urban knowledge graph construction service for the researchers in the urban computing domain. Input the urban text into the text box below to build the urban knowledge graph!

The online deployment demo of UrbanKGent. I'm the UrbanKG construction agent! What can I help you?

Input the urban text for relational triplet extraction, e.g., "Columbia University is a private Ivy league research university in New York City." Input urban entities for knowledge graph completion, e.g., "Entity 1: Columbia University, Point (40.8075, 73.9626)" and "Entity 2: Empire State Building, Point (40.7484, 73.9857)"

How to deloy UrbanKGent locally?

Open inferrence service locally via runing our opensourced code 


import uvicorn
from multiprocessing import Process
import argparse

def parse_args():
    parser = argparse.ArgumentParser()
    parser.add_argument("-host", type=str, default='0.0.0.0')
    parser.add_argument("-port", type=str, default=8000)
    parser.add_argument("-workers", type=int, default=1)
    return parser.parse_args()

if __name__ == '__main__':
    args = parse_args()
    uvicorn.run(app='open_llm_app:app', host=args.host, port=args.port, workers=args.workers)
Then, you access to UrbanKG construction service via following interface: 

curl -X POST "http://localhost:8000" /
